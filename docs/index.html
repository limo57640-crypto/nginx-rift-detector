NGINX Rift CVE-2026-42945 Detector

A read-only Bash self-check for NGINX servers. It reviews local version data, rewrite configuration, Linux ASLR, worker user, and log signals that need operator review.

Open repository Read the Ping7 guide Request repair help
Run this only on systems you own or are approved to review. The project does not include exploit code or instructions for unauthorized testing.

What it checks

  • NGINX version compared with fixed releases.
  • Rewrite rules that need manual review.
  • Long URI, encoding, and crash-log signals.
  • ASLR and worker privilege state.

Safe run path

Download the script, read it, then run it locally on the server.

curl -fsSLO https://raw.githubusercontent.com/limo57640-crypto/nginx-rift-detector/main/detect.sh
less detect.sh
sudo bash detect.sh

When to escalate

  • The result is VULNERABLE or SUSPICIOUS.
  • Worker crashes or unusual long request paths appear.
  • The server was patched late and public traffic was exposed.
